Android O更新

  • 简化了有关设置分组、标题和样式的指南。

之前的更新

2017年4月

重要更新

  • Material Design 颜色系统、可用性更新，以及用于创建调色板和测试可访问性的全新的颜色工具。
  • 文本字段：扩展了使用基础知识，并引入文本字段框以提高文本字段可发现性。
  • 双向性：添加了关于从右到左（RTL）图标的开发者指南的资源链接。

2016年12月

新增部分

  • 平台：讨论何时及如何将应用调整为Material Design，何时遵循平台惯例。
  • 应用快捷方式图标：包含创建主屏幕应用快捷方式图标的规范。
  • 帮助与反馈：描述如何展示、制作和放置帮助内容。

重要更新

  • 双向性：更新了关于在从右到左（RTL）语言中镜像 UI 内容和图标的指南。
  • 可访问性：新增关于声音、控件、标记元素和测试的指南。
  • 选择控件：包含了关于项目选择、切换和样式的详细示例。

2016年8月

新增部分

  • 通知：更新了 Android N 的通知，包括新的内容区域、交互、用户控制和短文本输入。
  • 小部件：提供用户参与、留存和获取的指南。
  • 确认和确知：涵盖何时以及如何确认用户操作。

重要更新

  • 导航：现在包括如何将前进和后退按钮作为导航策略一部分的详细信息。
  • 使用全屏模式进行Lean Back、沉浸模式和息屏交互。

2016年5月

新增部分

  • 动画：包括材料运动的原则、持续时间和缓动曲线、变换、编排和自定义动画的详细示例和规范。
  • 增长与沟通：提供关于用户参与、留存和获取的指南。
  • 可展开面板：描述用于创建流程和编辑内容的轻量级容器。

2016年3月

新增部分

  • 底部导航栏：允许在顶级视图之间切换。
  • 分屏：允许同时显示两个窗口（仅限Android）。

重要更新

  • 扩展可访问性指南，涉及颜色和对比度、声音、运动、层次结构等。
  • 如何在文本字段上显示密码输入。

2016年2月

重要更新

  • 扩展关于自定义应用配色方案的指南，并提供更多示例。
  • 对话框的错误状态、文本选择和文本字段标签的颜色规范。
  • 贴片现在新增文本字段颜色、月份和时间选择器以及底部工具栏（用于 Photoshop、Illustrator 和 Sketch）。

2015年12月

重要更新

  • 响应式 UI：增加了定位方式，和更多响应式的示例。
  • 文本字段：新增关于必填文本字段的说明。
  • Sketch 新增了线稿。
  • 标点符号：新增关于破折号和括号的指南。

2015年10月

新增部分

  • 步进器：描述不同类型的步进器如何通过数字编号来表示进度。

重要更新

  • 纸片：新增关于纸片移动和删除的指南。
  • 通知：包含了时间戳格式化指南。

2015年9月

重要更新

  • 导航：新增关于导航层次和结构、新导航方式和组合导航方式的指南。
  • UI 颜色应用：描述透明文本如何在不同背景颜色下保持可读性。
  • 按钮：包含了关于可切换按钮和扁平按钮行为的指南。
  • Tab 选项卡：包含了关于 tab 中文本大小的指南。

2015年8月

新增部分

  • 权限：包含了关于权限组、运行时权限、请求方式和如何处理被拒绝权限的指南。

重要更新

  • 底部卡片：包括持久显示的底部卡片、和模态化的底部卡片的描述、替代显示选项，以及在桌面和平板的显示。
  • 设置：包含了使用概述。

2015年7月

新增部分

  • 通知：包含为用户调整通知的指南，以及通知速览。
  • 指纹：提供关于将指纹作为身份验证选项的实施指南。

重要更新

  • 卡片：包含更多关于卡片 UI 控件、滑块和标签的图像示例。
  • 启动屏幕：包含实现占位符 UI 的指南。
  • 设置：关于标签和次要文本的额外指南。

2015年5月

新增部分

  • 自适应 UI：包括响应式布局指南、网格、断点。
  • 空状态：提供在常规内容无法显示时的状态指南。
  • 启动屏幕：包含品牌标志和 UI 占位内容。
  • 导航

重要更新

  • 海拔高度和阴影：从“3D空间中的对象”重命名，增加了关于海拔高度变化的新细节。
  • 仿真动画：更详细地描述了缓动术语。

2015年4月

新增部分

  • 数据表格
  • 单位和度量
  • 浮动操作按钮：已移至独立页面，增加了关于行为和过渡的新细节。
  • 应用结构
  • Adobe After Effects 的组件贴纸。
  • 数据截断和编辑
  • 设置中的写作指南。

重要更新

  • 排版：对密集和较高的语言的样式和行高提供进一步的指南。
  • 卡片：包含了更多关于操作和内容布局的规格。
  • 对话框：包含了额外的布局指南。
  • Tab 选项卡：增加关于标签内容和更多尺寸规格的指南。
  • 滚动技术：增加关于重叠内容的指南。
